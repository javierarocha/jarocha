== CHANGE LOG ==
= Version 2.0.1 =
* Excerpt Option bug fixed.
* Multicheck Option to Hide categories in Blog page issue is fixed.

= Version 2.0 =
* Theme Options migrated to customizer as per new WordPress theme review guidelines.

= Version 1.0.2 =
* Removed deprecated function for WP_Widget class and used the __construct() instead as per WordPress 4.3
* One page toggle menu bug fixed.
* Slider Navigation issue in chrome fixed.
* Styling adjustment for single post navigation is fixed.
* Menu alignment in small device (when header Text and Logo are not used) is fixed.
* Small adjustment in Breadcrumb NavXT plugin support code.
* Added the 'updated' class for Google Structured Data.
* Added Featured image options in TG: Service widget.

= Version 1.0.1 =
* Console Error: 'ample_slider_value is not defined' is fixed.
* Default WordPress Gallery Supported.
* Option added to make Service widget Title and Icon unclickable.
* Height of Call To Action widget is adjusted (if no background image is used).

= Version 1.0 =
* All brand new.
